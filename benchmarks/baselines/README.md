# Approved baselines

Only full results captured by the dedicated `ram-benchmark` runner may be
placed here. Numeric data is intentionally absent until that external runner
has produced repeated measurements and a reviewer has approved one candidate.

See [the benchmark procedure](../README.md) for the runner contract and the
required review metadata. Local or smoke results must never be committed as an
approved baseline.

---

# 已批准基线

这里只能存放专用 `ram-benchmark` runner 捕获的完整结果。在外部 runner 产生重复测量且审阅者
批准一个候选前，仓库有意不提供数值数据。

runner 契约和必需审阅元数据见[基准流程](../README.md)。本地或烟测结果绝不能作为已批准
基线提交。
